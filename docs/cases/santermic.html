Сантермик

Mагазин инженерной сантехники на окраине Москвы с опытом работы 17 лет и базой товаров более 40 000.

Задача

Увеличить объем продаж магазина за счет реализации торговой онлайн–площадки и ее продвижения.

Наше Решение

Интернет-магазин на связке Wordpress +
WooCommerce

Анализ конкурентов на ключевые фразы и видимость в поиске по релевантным запросам.

Настройка динамических фильтров по каталогу для ускорения работы сайта.

Добавление технических страниц: Оплата, Доставка, Гарантии, Контакты, Акции, Обмен/ возврат, для поднятия позиций в поисковых ситемах.

Релевантные запросы

SEO Оптимизация

Составление семантического ядра на основе маржинальных категорий товаров.

Кластеризация семантики по релевантным страницам.

Оптимизация мета–информации на сайте.

Оптимизация заголовков страниц и текстов описания товаров.

Переходы

Продвижение и реклама

Запуск внешней оптимизации для постепенного наращивания ссылочной массы.

Регистрация торговой площадки в тематических каталогах, на форумах и сервисах.

Создание таргетированной рекламной кампании в Яндекс.Директ. Сегментирование аудитории по целевым запросам/интересам/сайтам конкурентов.

Создание рекламной кампании в системе Яндекс.Маркет. Запуск дополнительного канала привлечения трафика и продаж.

Регистрация сайта в экзотических поисковых системах.

Отбор площадок–агрегаторов для размещения PR статей магазина. Заказ статей через систему Sape для повышения индекса цитирования.

История показов по прямому запросу

Результат

За год работы, мы реализовали динамику роста показов веб-ресурса с 0 до 15 тыс. (декабрь 2016).

Поиск по названию магазина "Сантермик" поднялся до ~150 запросов в месяц. Магазин вышел на стабильный показатель ~100 ежедневных лидов.

Общий оборот магазина вырос в 2,2 раза.
